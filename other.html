PolterGuard


Summary

This game is a small group project made by six students, including me. The Game Pie project is the first project of the Semester 4 Game Design specialization at Fontys University of Applied Sciences. The goal was to create a first-person shooter game within four weeks, with each week focusing on a different topic.

  • Week 1: Introduction week
  • Week 2: Game analysis
  • Week 3: Game design
  • Week 4: Game development

Our idea was to create a horror-themed shooter set in a tech store. The player takes on the role of a security guard when the store is suddenly possessed by a poltergeist.

Responsibilities

  • Game design
  • Shooting mechanic
  • Menu navigation (Main menu, pause menu, death screen, etc.)
  • Second floor realization
  • Git "master"
  • User testing
